To run one of these python programs, one must open command prompt and type the following command:

py [Corresponding Program Name].py


The version of python I used is Python 3.7


The following programs are available:
------------------------------------------------------------------------------------------------------------------
Unit_Conversion.py

A program that converts from miles per hour to kilometers per hour. The user may input an initial
mph value.
------------------------------------------------------------------------------------------------------------------
Slope.py

A program that returns the sum of digits given by the user.
------------------------------------------------------------------------------------------------------------------
Hello_World.py

This program greets the user with "Hello!" It then proceeds to ask the user questions regarding their name, favorite
color, and age. It then returns a statement repeating the information obtained in these questions.
------------------------------------------------------------------------------------------------------------------
Fibonacci.py

This program prints the nth number in the Fibonacci Sequence.
------------------------------------------------------------------------------------------------------------------
Convertor.py

This program prompts the user for an initial Fahrenheit temperature to convert to Celsius. It then prints
this converted value 5 times.
------------------------------------------------------------------------------------------------------------------
Cash.py

This program prompts the user for the amount of change they need returned. This program uses a greedy
algorithm to determine the least amount of coins needed to return to satisfy the amount needed. This program 
uses USD, and thus the coins used to determine change are:

Quarters: $0.25
Dimes: $0.10
Nickels: $0.05
Pennies: $0.01
